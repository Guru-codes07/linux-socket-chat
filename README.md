# 💬 Linux Socket Chat

A TCP-based client-server chat application built in C using Linux socket programming. This project demonstrates the fundamentals of TCP/IP networking, socket APIs, and client-server communication on Linux systems.

## 🚀 Features

* TCP socket communication
* Client-server architecture
* Real-time message exchange
* Terminal-based interface
* Built and tested on Linux

## 🛠️ Technologies Used

* C Programming
* Linux
* TCP/IP
* POSIX Socket APIs
* GCC

## 📂 Project Structure

```text
linux-socket-chat/
├── client.c
├── server.c
├── Makefile
├── README.md
└── screenshots/
```

## 🔨 Build

```bash
gcc server.c -o server
gcc client.c -o client
```

## ▶️ Run

Start the server:

```bash
./server
```

Start the client:

```bash
./client
```

## 🎯 Learning Objectives

This project was developed to gain hands-on experience with:

* Linux Socket Programming
* TCP/IP Networking
* Client-Server Architecture
* System Programming in C
